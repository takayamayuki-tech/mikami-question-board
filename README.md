# みかみグルコン 質問回答シート

みかみさん専用の「グルコン質問」閲覧・回答管理アプリ。
Google フォーム（→スプレッドシート）に届いた質問を見やすいカードUIで表示し、
「回答済み」ボタンで状態管理ができる。

## 主な機能

- **自動連携**: スプレッドシート（Google フォーム回答先）と直結。新しい質問は自動反映
- **見やすいカードUI**: 受講生名 / 回答日 / 本名 / 質問内容 / Addness使用可否 を一目で確認
- **未回答が一番下**: 最新の未回答が下に積まれる（古いものが上）
- **回答済みボタン**: クリックでスプレッドシートに状態を書き戻し → 灰色＋区切りの下に移動
- **2026年4月以降のみ表示**: 古い質問はフィルタリング
- **PC / スマホ両対応**: レスポンシブデザイン
- **共有はURL1本**: GAS Web App として公開、URLを送るだけ
